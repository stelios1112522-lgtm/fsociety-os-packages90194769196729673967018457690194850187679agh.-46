# FSOCIETY OS Package Repository

This repository contains binary packages for FSOCIETY OS, a custom Void Linux distribution.

## Repository Structure

- `x86_64/` - Binary packages (.xbps files) for x86_64 architecture
- `x86_64-repodata` - Repository metadata (generated by xbps-rindex)
- `fsociety-os-pubkey.asc` - GPG public key for package verification

## Adding This Repository to Void Linux

Add the repository to your Void Linux system:

```bash
sudo mkdir -p /etc/xbps.d
echo "repository=https://raw.githubusercontent.com/YOUR_USERNAME/fsociety-os-packages/main" | sudo tee /etc/xbps.d/fsociety.conf
```

Import the GPG key:

```bash
sudo xbps-importkey fsociety-os-pubkey.asc
```

Sync and install packages:

```bash
sudo xbps-install -S
sudo xbps-install -S <package-name>
```

## Package Management

Packages are signed with the GPG key identified by `packages@fsocietyos.com`.
